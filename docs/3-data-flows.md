### Gaji Calculation Algorithm

```
STEP 1: Determine shift-based gaji pokok (sesuai dengan crew's status_gaji = PRO)

Day Check:
â”œâ”€ Monday-Thursday â†’ Weekday Full Time â†’ 75,000
â”œâ”€ Friday-Sunday + Shift 1 (9-15) â†’ 35,000
â”œâ”€ Friday-Sunday + Shift 2 (15-21) â†’ 35,000
â””â”€ Friday-Sunday + Full Time â†’ 100,000

STEP 2: Calculate keterlambatan potongan

IF jam_masuk > jam_masuk_official + 10 minutes THEN
  menit_terlambat = EXTRACT(MINUTE FROM (jam_masuk - jam_masuk_official))
  kelipatan_10menit = FLOOR(menit_terlambat / 10)
  potongan = kelipatan_10menit * 5000
ELSE
  potongan = 0
END IF

Example:
â”œâ”€ Masuk jam 12:05 (target 12:00) â†’ 5 min, < 10 min â†’ potongan = 0
â”œâ”€ Masuk jam 12:15 (target 12:00) â†’ 15 min, 1x kelipatan â†’ potongan = 5,000
â”œâ”€ Masuk jam 12:27 (target 12:00) â†’ 27 min, 2x kelipatan â†’ potongan = 10,000
â””â”€ Masuk jam 12:45 (target 12:00) â†’ 45 min, 4x kelipatan â†’ potongan = 20,000

STEP 3: Calculate bonus tim

omset_daily = SUM(all transactions today)
is_weekday = (day = Monday-Thursday)

target_weekday = 1,000,000
target_weekend = 1,500,000
base_bonus = 20,000
bonus_per_tier = 5,000
tier_amount = 50,000

IF is_weekday AND omset_daily >= target_weekday THEN
  bonus = base_bonus + FLOOR((omset_daily - target_weekday) / tier_amount) * bonus_per_tier
ELSE IF NOT is_weekday AND omset_daily >= target_weekend THEN
  bonus = base_bonus + FLOOR((omset_daily - target_weekend) / tier_amount) * bonus_per_tier
ELSE
  bonus = 0
END IF

Example:
â”œâ”€ omset = 1,050,000 (weekday target) â†’ bonus = 20,000 + FLOOR(50/50) * 5000 = 25,000
â”œâ”€ omset = 1,200,000 (weekday target) â†’ bonus = 20,000 + FLOOR(200/50) * 5000 = 40,000
â””â”€ omset = 2,000,000 (weekend target 1,500k) â†’ bonus = 20,000 + FLOOR(500/50) * 5000 = 70,000

STEP 4: BONUS distribution (dibagi rata ke semua crew yang hadir)

total_bonus_pool = CALCULATED bonus_tim (step 3)
crew_hadir_count = SELECT COUNT(DISTINCT crew_id) FROM attendance WHERE tanggal = today AND jam_keluar IS NOT NULL
bonus_per_crew = total_bonus_pool / crew_hadir_count

STEP 5: Gaji Bersih

gaji_bersih = gaji_pokok - potongan + bonus_per_crew

STEP 6: Generate Slip & Update Attendance

UPDATE attendance SET
  gaji_bersih = calculated value,
  status = 'FINALIZED'
WHERE id = attendance_id;

INSERT INTO slip_gaji (crew_id, tanggal, content_html)
VALUES (...);

UPLOAD slip to Storage:
  Path: slip_gaji/{crew_id}/{tanggal}.pdf
```
